Control your desktop monitor brightness just like on a MacBook using the brightness keys on an Apple keyboard.

Your monitor needs to support DDC/CI for this app to work.

https://github.com/tomun/NativeDisplayBrightness

Modified from the original NativeDisplayBrightness to respond to the brightness keys by Tom Underhill.

Uses DDHotKey Copyright (c) Dave DeLong

Uses code borrowed from ddcctl by Joey Korkames which uses code from DDC-CI-Tools-for-OS-X by Jonathan Taylor
